BNL Summer Internship

In the summer 2021, I worked under with Brookhaven National Lab under Dr. Atiya Banerjee. My project consisted of generating mononuclear and dinuclear metal active sites in proteins using computational chemistry data. This consisted of building a library of Python files that pulled this information and generated modeling files for specific molecular geometries in these active sites.

SkateStop

SkateStop is a website designed to bring information on where to skate to the skateboarding community. Implemented using the Google Maps API, it allows users to store information about their favorite places to skate and share it with other users. This website is being designed and implemented as a final project for a website development class. Code is available for view by the public.
